# Red-Black Tree
## 특징
_1. 각 노드는 빨 혹은 검의 색을 가진다_  
_2. bst의 형태를 가진다_  
## 룰
_1. 루트는 항상 흑색이다_  
_2. 빨강색은 연속으로 2번 올 수 없다(부모 자식 다 빨강)_  
_3. 리프 노드는 흑색으로 규정한다_  
_4. 루트에서 리프까지 흑색 노드의 수는 모두 같다_  
## 2번 위반의 경우들  
### Case1
부모, 자식, 삼촌 모두 빨강 -> 조부모부터 색을 검 빨 검으로 recolering  
```
              'R15'
        'R14'       
    B12
        'R10'
B8
```
### Case2  
부모 자식만 빨강 -> 특정 노드들을 회전시켜줌  
### Case2-1
한쪽으로 치우친 경우 -> 조부모, 부모, 자식 세 노드를 치우친 방향 반대로 회전
```
//B12, R14, R15 좌로 회전
                  K
              R15
                  Z
        R14
              Y
  B12

        X
```
```
//자리에 맞게 recoloring           
              K
        R15
              Z
  B14
              Y
        R12
              X
```
### Case2-2
좌우 혹은 우좌의 형태인 경우 -> 부모, 자식 두 노드를 치우친 방향 반대로 회전  
                           -> 조부모, 부모, 자식 세 노드를 치우친 방향 반대로 회전  
                             
예) 조부모부터 우좌면 2개 노드를 우로 회전 후 3개 노드를 좌로 회전 
```
//R14, R15를 우로 회전
              K

        R15
                    Z
              R14
                    Y
  B12

        X
```
```
//B12, R14, R15 좌로 회전
                    K
              R15
                    Z
        R14
                    
              Y
                    
  B12

        X
```
```
//자리에 맞게 recoloring
              K
        R15
              Z
  B14
              Y
        R12
              X
```
